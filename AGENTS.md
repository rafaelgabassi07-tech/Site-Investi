# Agent Instructions

- Sempre que arquivos de testes (como scripts temporários `test_*.js`, JSONs ou arquivos HTML de saída) forem criados para depuração ou prototipação no backend, lembre-se de deletá-los (`delete_file`) do diretório raiz assim que cumprirem seu propósito. O código fonte (workspace) deve ser mantido sempre limpo e livre de sujeiras temporárias.
